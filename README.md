🚀 KB to Byte Converter - Java
Welcome to the KB to Byte Converter project! This is a simple Java program designed for beginners to understand basic input/output and mathematical operations in Java.

📝 Description
This program takes a value in Kilobytes (KB) from the user and converts it into Bytes. It's a fundamental exercise for ICT students to learn how data storage units work.

🛠️ How it Works
The conversion formula used is:

1 KB = 1024 Bytes

💻 How to Run
Make sure you have Java JDK installed.

Download the Converter.java file.

Open your terminal or VS Code.

Compile the file:

Bash
javac Converter.java

5. Run the program:
   ```bash
   java Converter

Simple and clean code.
